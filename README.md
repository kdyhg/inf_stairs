# INF STAIRS

키보드로 좌우 계단을 빠르게 올라가는 20초 제한 레트로 계단 게임입니다. 원작 에셋을 복제하지 않고, 픽셀풍 분위기와 단순한 좌우 계단 규칙을 바탕으로 새로 만든 구현입니다.

## 기능

- 시작 전 닉네임 입력 필수
- 좌우 방향키로 한 칸 이동
- `Shift` + 좌우 방향키로 같은 방향 계단을 두 칸 이동
- 다음 계단 방향은 랜덤 생성
- 방향을 틀리면 즉시 종료
- 제한시간 20초
- 올라간 층수가 점수
- 랭킹은 브라우저 로컬 저장소가 아니라 서버의 `data/rankings.json`에 저장
- 새로고침 버튼으로 서버 랭킹 재조회

## 실행

```bash
npm start
```

브라우저에서 `http://localhost:3000`을 열면 됩니다.

## 배포 메모

GitHub Pages 같은 정적 호스팅만으로는 서버 랭킹 저장이 동작하지 않습니다. 랭킹을 여러 사용자와 공유하려면 Node 서버가 파일을 쓸 수 있는 환경에서 실행하거나, Supabase/Firebase/Redis 같은 외부 데이터 저장소를 연결해야 합니다.
